# Adcelerate Diagnose Mode

Diagnose system failures, identify root causes, propose and apply fixes, and record patterns for future reference.

## Trigger

Use this skill when:
- A registered system is failing or producing errors
- Output quality has degraded
- A system dependency has changed or broken
- The engineer wants to investigate system health

## Usage

```
/adcelerate-diagnose autocaption
/adcelerate-diagnose The storyboard system is producing blurry images
```
